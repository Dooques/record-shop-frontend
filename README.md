# Record Shop Frontend

This is the repository for the frontend portion of the Record Shop Project

This app is designed to use the Blazor front end library from .NET to create a dynamic website which receives data from the ASP.NET backend

Features:
- Blazor Framwork uses Razor pages, combining C# code and HTML to create a beautiful user experience
- Use Http Client to retrieve data from the backend API
- Use Linq queries to filter the received data, updating the web pages dynamically
- Use Blazor forms to update and create new data which will update the backend through API endpoints
- Use a mix of vanilla CSS and Bulma CSS to create the UI
